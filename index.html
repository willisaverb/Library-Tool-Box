Welcome to the Library Staff Tools Sandbox

This site is a collection of simple tools designed to help public library staff with their daily tasks. Explore the tools using the navigation bar above.

Potential Future Tools

Here are some ideas for other tools that could be developed to benefit library staff:

  • Program Attendance Tracker: A tool to easily record and visualize attendance for library programs.
  • Patron PC Booking System: A simple interface for managing public computer reservations.
  • New Acquisitions Showcase Generator: A tool to quickly create a webpage or flyer for new books and materials.
  • Community Resources Directory: A searchable directory of local community services for referral.
  • Inter-Library Loan Request Manager: A streamlined way to manage incoming and outgoing loan requests.
